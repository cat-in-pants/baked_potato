Консольная программа на C++ для управления личными финансами

Основные функции:

1. Добавление записей о доходах и расходах (категория, сумма, дата, описание)
2. Просмотр с фильтрацией (все/доходы/расходы/по дате) и сортировкой
3. Удаление записей по ID
4. Статистика: общие доходы/расходы, баланс, средние траты в месяц, разбивка по категориям
5. Автосохранение в файл finance_data.txt после каждого изменения
6. Загрузка данных из пользовательского файла

Технические особенности:

1. Данные хранятся в текстовом файле (формат: ID|Тип|Категория|Сумма|Дата|Описание)
2. Поддержка русского языка в консоли Windows (UTF-8)
3. Корректная обработка кириллицы при отображении таблиц
4. Проверка корректности вводимых данных
