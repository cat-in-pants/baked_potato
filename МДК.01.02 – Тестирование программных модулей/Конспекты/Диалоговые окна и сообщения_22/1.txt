Диалоговое окно — это отдельное окно интерфейса, которое:

Взаимодействует с пользователем
Требует реакции
Обычно блокирует основное окно

Диалоги используются для:

Показа сообщений
Подтверждения действий
Запроса данных
Сообщения об ошибках

Виды модальных окон

1. Модальное окно:
   Блокирует основное окно
   Пользователь обязан отреагировать
   Пример: подтверждение удаления, окно авторизации
2. Немодальное окно:
   Не блокирует работу основного окна
   Можно игнорировать
   Пример: справка, окно настроек

Qt6 предоставляет готовые диалоговые компоненты:

QMessageBox — для показа сообщений
QInputDialog — для простого ввода одного значения
QFileDialog — для работы с файлами (открыть, сохранить, выбрать директорию)
QDialog — для создания пользовательских диалогов

QDialog — собственные диалоговые окна

Используется, когда:

Форма сложнее одного поля
Нужна кастомная логика
Требуется несколько кнопок и полей
Диалоговое окно является отдельным классом со своей логикой и возвращает результат.

Как показывать диалог

exec() — блокирует основное окно и возвращает результат закрытия (для модальных окон)
show() — не блокирует окно (для немодальных)

Жизненный цикл диалога

1. Создание объекта
2. Отображение (exec() или show())
3. Взаимодействие с пользователем
4. Закрытие
5. Уничтожение

Важно:

Не держать диалоги активными без необходимости
Не хранить состояние в диалоге дольше, чем нужно
