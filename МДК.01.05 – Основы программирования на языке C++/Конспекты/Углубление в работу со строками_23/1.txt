1. Основные характеристики std::string

std::string - класс, управляющий памятью автоматически.
Хранит длину строки.
Поддерживает безопасные операции


2. Копирование строк

Каждое копирование std::string выделяет новую память и копирует символы
Рекомендация:
  Передавать строки по const std::string&
  Аккуратно возвращать строки


3. std::string_view

Не владеет строкой, только предоставляет доступ к ней
Преимущества:
  Нет копирования.
  Работает с std::string, char*, строковыми литералами


4. Поиск в строках

Используется метод find().
std::string::npos - специальное значение (обычно -1 в виде size_t), означающее "не найдено"


5. Сравнение строк

Строки можно сравнивать операторами ==, !=, <, > и т.д
Сравнение происходит лексикографически


6. Извлечение подстроки

Метод substr(pos, count) возвращает новую строку (копирует)


7. Разделение строки

Можно разделять вручную с помощью find() и substr()
Оптимально использовать string_view для уменьшения копирований


8. Преобразование строк ↔ числа

std::stoi, std::stod и др. - преобразуют строку в число
std::to_string - преобразует число в строку
Могут выбрасывать исключения:
  std::invalid_argument
  std::out_of_range


9. Форматирование строк

Используется std::stringstream для сложного форматирования


10. Когда что использовать

   Ситуация             Что использовать
Владение строкой          std::string
Только чтение           std::string_view
Разбор текста       find, substr, string_view
Частые копии           const std::string&
