1. Библиотека для работы с файлами

#include <fstream>


2. Основные классы

  Класс        Назначение
ifstream     Чтение из файла
ofstream      Запись в файл
fstream      Чтение + запись

3. Открытие файла

Чтение
std::ifstream fin("file.txt");

Запись
std::ofstream fout("file.txt");

Чтение и запись
std::fstream f("file.txt");


4. Проверка открытия файла

if (!fin.is_open()) {
}


5. Режимы открытия файла

  Флаг             Что делает
ios::in              Чтение
ios::out             Запись
ios::app        Дописывать в конец
ios::trunc        Очистить файл
ios::binary       Бинарный режим


6. Чтение из файла

Чтение слов/чисел
int x;
while (fin >> x) {
}

Чтение строки
std::string line;
while (std::getline(fin, line)) {
}


7. Запись в файл

fout << "Текст" << value << std::endl;


8. Закрытие файла

fin.close();


9. Где программа ищет файлы

В рабочей директории
По указанному пути

10. Основные операции

      Операция                   Как делать
Открыть для чтения        ifstream fin("file.txt")
Открыть для записи        ofstream fout("file.txt")
Проверить открытие             fin.is_open()
Чтение строк                getline(fin, line)
Чтение чисел                   fin >> number
Запись данных                  fout << data
Закрыть файл                    fin.close()


11. Типичные задачи

1. Подсчет строк - читать файл через getline(), считать строки
2. Поиск самой длинной строки - сравнивать длины строк
3. Копирование файла - читать построчно, записывать в другой файл
4. Обработка символов - читать посимвольно через fin.get(ch)
5. Чтение чисел - использовать fin >> number

12. Важные моменты

Всегда проверять, открылся ли файл
getline() - основной способ чтения текстовых файлов
Поток автоматически закрывается при уничтожении объекта
При ошибках чтения цикл while (fin >> x) прекращается
