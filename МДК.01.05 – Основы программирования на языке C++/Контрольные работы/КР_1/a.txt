1. Массив - статический размер на стеке. Vector - динамический размер в куче, сам управляет памятью, имеет удобные методы.

2. string_view - только для чтения, не владеет памятью (быстрый, легкий). string - для хранения и изменения строк.

3. system_clock - реальное время (может корректироваться). steady_clock - монотонный, только для измерения интервалов.

4. Исключения - для критических ошибок. optional - когда отсутствие значения - штатная ситуация.

5. Препроцессинг - Компиляция - Ассемблирование - Линковка
