Техническое задание: Prismix

Приложение для создания аддонов и ресурс-паков для Minecraft Bedrock Edition без программирования


 Базовая структура
1. Сплэш-скрин с анимацией
2. Главное окно с боковым меню
3. Навигация между экранами
4. Кнопка "Назад" для возврата на главный экран

 Экраны
1. Главный экран: фраза, поиск, карточки "Новый проект" и "Шаблоны", секция "Последние проекты"
2. Экран категорий: 4 карточки (Блоки, Предметы, Сущности, Другие текстуры)
3. Боковое меню: пункты "Новый проект", "Шаблоны", "Импорт", "Настройки", "О приложении"

 Экраны категорий
1. Экран "Блоки": подкатегории (Строительные, Природные, Декоративные, Функциональные, Редстоун), сетка текстур, поиск
2. Экран "Предметы": подкатегории (Броня, Инструменты, Оружие, Еда, Зелья, Яйца спавна, Ингредиенты, Разное), сетка текстур, поиск
3. Экран "Сущности": подкатегории (Верхний мир, Незер, Энд, Кастомные), сетка текстур, поиск
4. Экран "Другие текстуры": подкатегории (Окружение, Прочее, Картины, Частицы, Цветокарта, Карта), сетка текстур, поиск

 Функционал
1. Отображение текстур из папок ассетов
2. Пиксельное отображение текстур (без сглаживания)
3. Поиск по названию текстуры в реальном времени
4. Редактор текстуры: загрузка/замена изображения, предпросмотр
5. Классификатор текстур: логическое разделение на подкатегории
6. Генерация структуры папок для аддона/ресурс-пака
7. Экспорт проекта в правильную структуру Bedrock
8. Импорт существующих проектов
9. Система шаблонов
10. Настройки приложения

 Стилизация
1. Морская цветовая схема
