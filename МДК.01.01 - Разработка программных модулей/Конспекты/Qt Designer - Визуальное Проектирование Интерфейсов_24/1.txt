Qt Designer
Инструмент для визуального проектирования GUI (графического пользовательского интерфейса)
Позволяет создавать интерфейсы путем перетаскивания элементов
Сохраняет результат в формате .ui (XML-структура)
Может быть преобразован в Python-код с помощью pyuic6

Типы форм

MainWindow — главное окно приложения
Dialog — диалоговое окно
Widget — самостоятельный виджет

Виджеты

Стандартные элементы: QLabel, QPushButton, QLineEdit, QListWidget, QComboBox
Настраиваемые свойства: текст, размеры, шрифт, цвета

Layouts

QVBoxLayout — вертикальное расположение
QHBoxLayout — горизонтальное расположение
QGridLayout — табличное расположение
Обеспечивают автоматическую адаптивность при изменении размеров окна


Механизм Signals/Slots
Система реакции на события (сигналы) через слоты

Promote to…
Возможность использования пользовательских виджетов в .ui файле


Способы работы с .ui файлами

Способ 1: Конвертация в Python
Создание Python-класса с методом setupUi()
Преимущество: скорость выполнения

Способ 2: Динамическая загрузка
Загрузка UI во время выполнения через loadUi()
Преимущество: гибкость, возможность изменять интерфейс без перекомпиляции


Преимущества

Быстрое прототипирование интерфейсов
Визуальное проектирование без написания кода компоновки
Интуитивное редактирование размеров и расположения элементов

Недостатки

Сгенерированный код сложен для ручного редактирования
Ограниченная функциональность без дополнительного программирования
Сложная логика требует написания Python-кода отдельно
Риск создания неэффективных интерфейсов при неправильном использовании layouts
